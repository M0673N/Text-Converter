## Deployment

## Running locally

Open a terminal and run the following code:
For Windows:
```
git clone https://github.com/M0673N/Text-Converter.git &&
cd ./Text-Converter &&
start index.html
```
For Linux distros:
```
git clone https://github.com/M0673N/Text-Converter.git &&
cd ./Text-Converter &&
xdg-open index.html
```
